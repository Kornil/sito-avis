# Avis Rovigo website
Currently in development

## Installation
Run `npm install`

## Develop
1. Run `weback -watch`
2. Run `npm run dev` then open a browser to [localhost:3000](http://localhost:3000)

## Test
Run `npm run test`

## Build
Run `npm run build`

## Features
* Express
* React with Redux
* React Router
* Firebase
